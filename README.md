## 프로젝트 주요내용
웹 상의 책꽂이에 자신이 읽은 책을 배치하고 꾸며 공유하는 도서 기반의 sns입니다.
좋아하는 책을 서점에서 사고 채워진 책꽂이를 보며 느끼는 뿌듯함을 느낍니다.
내가 읽은 책을 공유하고 싶고 정리하고 싶은 욕구를 이 서비스를 통해 도움을 줄수있습니다.
